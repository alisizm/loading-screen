## 🚀 Installation

1.  Acquire the script from the Tebex link above.
2.  Download the resource from your Cfx.re Keymaster granted assets.
3.  Extract the folder into your server's `resources` directory (e.g., `[standalone]/alisizm_loadingscreen`).
4.  Add `ensure alisizm_loadingscreen` to your `server.cfg`.
5.  Restart your server.

### 🖌️ How to Customize
You can easily edit the visuals and text.
1.  Navigate to the `html` folder within the resource.
2.  Replace `logo.png` with your own logo (ensure the filename remains the same or update the HTML).
3.  Replace the background image in the `images` folder.
4.  Edit `index.html` to change text elements and server name.
